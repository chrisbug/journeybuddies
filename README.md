# journeybuddies

## What is it for.
---
JourneyBuddies is a group travel application that will
help clients enjoy a more stress free holiday.
#### Key features
* Messaging service for users.
* Integration with Google maps for navigation.
* Task organizer for whole group.
* Photo sharing.
